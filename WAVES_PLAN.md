# Future Waves Plan

Roadmap for adding events to the curated dataset, with the strict
EDITORIAL.md rule 8 (Sunni 4-madhhab orthodoxy) and the ≥2-classical-Sunni-source bar enforced. Drafted 2026-04-29 at 1,207 events / 249 of 366 calendar days covered.

This is a living document. Every entry is a wave I can confidently produce
under the editorial bar with concrete content (not a vague theme).

## Where I see the dataset stopping

The honest stopping points, in order:

1. **~1,500 events / 280-300 days covered** — the natural ceiling for the
   well-attested classical Sunnī biographical corpus when filtered through
   the strict orthodoxy + ≥2-source bar. Beyond this, each new wave
   produces fewer day-precise events and more single-source year-only
   entries.
2. **~1,800 events** — the absolute limit if we go heavy on modern
   institutional foundings, hadith-collection printing milestones, and
   late-Ottoman-Egyptian biographical-tradition third-tier figures. Past
   this, the bar would have to drop or the figures become too obscure to
   defend.
3. **Calendar saturation (366/366 days)** is **not** a target. About 25-40
   days will likely remain "lesson-only" because the classical sources
   simply don't day-attest enough events to anchor each one. The dateless
   `lessons/` corpus already fills these gaps.

The frontier is **honesty about classical attestation**. Pushing past it
would corrode what makes the dataset valuable.

## Out of scope — what I will NOT add

- **Anything Mouride / Tijaniyya / Tijani-Niassene / Akbarian / executed-for-heresy / Twelver-doctrinal / Mu'tazilī-as-theologian.** Already exhaustively listed in EDITORIAL.md rule 8.
- **Living scholars.** No death event without a death. Wait for posterity.
- **Speculative early-Islamic events without ≥2 classical Sunni sources.** A vague "tradition holds X happened" doesn't meet the bar.
- **Modern political-Islamic-movement leaders where their religious significance is contested by mainstream Sunni scholarship.** Case-by-case under rule 8.
- **Pre-Hijra figures other than the Prophets and the very small set already covered** (the Year of the Elephant, etc.). Pre-Islamic Arabia is out of scope.
- **Indigenous-conversion myths from the Muslim periphery** (some West African / Southeast Asian / Caucasus founding-of-X stories where date is not attested in any classical source).
- **Late-Ottoman / modern Turkish Sufi orders with controversial claims** — Süleymancılar (Süleyman Hilmi Tunahan founded), Iskenderpaşa Cemaati, Hizmet movement (Gülen). All criticised by mainstream Sunni Turkish scholars; default drop per rule 8.
- **Modernist reformist controversies that fail rule 8** — even mainstream-named figures like Tāhā Ḥusayn (Egyptian modernist who fell out with al-Azhar) or Naṣr Ḥāmid Abū Zayd (declared apostate by Egyptian court). Drop.

## Cadence

When dispatching new waves: 6 parallel agents per batch, each on a distinct
topic cluster from this list, with the rule-8 prompt embedded explicitly.
After integration, re-run pipeline.build / pipeline.validate.

After ~6 more batches at ~70 events per batch, expect to plateau at ~1,500
events. Past that, consider switching to:

1. **Day-precision audits** — re-check existing year-only events for any
   classical day-attestation we missed.
2. **Hadith-collection completion events** — only where dates are
   attested.
3. **Lesson-corpus expansion** — fill remaining empty calendar days with
   dateless `lessons/` content.
4. **Multilingual polish** — review ar/fr translations for consistency.
5. **Source-URL cleanup** — replace remaining Wikipedia URLs with
   academic / EI2 / Sunnah.com permalinks where available.

This file is the master plan. Update it when waves are dispatched (mark
done) or when new candidates emerge from primary-source reading.
